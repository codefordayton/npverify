Nonprofit Verifier

TL;DR - Spend less than 5 minutes to check if a Dayton area nonprofit needs volunteers!

{% if nonprofit %} Try one! {% endif %}

Code For Dayton is partnering with Dayton Serves to make it easier to match nonprofits with volunteers. There are literally thousands of nonprofits in the Dayton area. A small percent of them want external volunteers. The primary purpose of this site is to find them so that we can highlight them.

A secondary purpose of the site is to find where Code For Dayton can help. We want to identify nonprofits that need help with websites and social media.

The process of verifying a nonprofit is simple. You'll perform a search and capture your results. Most nonprofits will probably take a minute or less.

Thank you in advance for your help!
